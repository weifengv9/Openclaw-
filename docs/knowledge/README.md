# 知识库总览

> 7步决策流的智慧沉淀中心

## 目录结构

```
knowledge/
├── README.md         — 本文件，知识库索引
├── decisions/        — 决策记录（#4 决策产出）
├── plans/            — 筹划方案（#2 筹划产出）
└── lessons/         — 经验教训（DocGardener 总结）
```

## 流转规则

```
总办接收指令
    │
    ▼
筹划生成方案  →  存入 plans/
    │
    ▼
商讨+决策     →  存入 decisions/
    │
    ▼
贯彻执行
    │
    ▼
DocGardener 沉淀经验  →  存入 lessons/
```

## 知识库维护

- **扫描周期**：每周三 9:00（DocGardener 自动执行）
- **全面检查**：每月一次
- **触发条件**：每次决策流程完成时自动沉淀

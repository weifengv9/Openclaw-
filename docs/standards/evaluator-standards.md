# Evaluator Agent 验收标准

> Evaluator = 评估器，负责质量审查、打分、反馈

---

## 四维度打分表

每个维度 1-10 分，最低达标线 7 分

| 维度 | 评估要点 | 达标线 |
|------|----------|--------|
| **质量** | 准确、完整、无错误 | ≥ 7 |
| **原创性** | 独特价值、无重复 | ≥ 7 |
| **工艺** | 结构清晰、格式规范 | ≥ 7 |
| **功能** | 满足需求、可落地 | ≥ 7 |

**综合评分** = (质量 + 原创性 + 工艺 + 功能) / 4

**综合达标线**：≥ 7 分

---

## 评估流程

1. **阅读生成内容**
2. **对照四个维度打分**
3. **给出具体反馈**（优点 + 改进点）
4. **综合判定**：通过 / 需修正

---

---

## 评估要点

- 公正客观，不带个人偏好
- 具体指出问题所在，不说"不太好"这种模糊的话
- 给出可执行的修正建议
- 关注核心功能是否实现，不纠结细枝末节
